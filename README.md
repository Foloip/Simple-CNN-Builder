

# SCB: Simple CNN Builder

Простой инструмент для создания и тестирования сверточных нейросетей. Позволяет собирать архитектуры CNN, обучать их на датасетах MNIST и CIFAR-10, а также тестировать на своих изображениях. Предназначен для новичков в машинном обучении.

**Важно**: это консольное приложение (CLI), всё взаимодействие происходит через терминал.

---

## Требования
- Python 3.9+
- TensorFlow 2.13+
- 4 ГБ оперативной памяти
- 2 ГБ свободного места

---

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Foloip/Simple-CNN-Builder.git
cd Simple-CNN-Builder
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```


---

## Использование

### Запуск приложения
```bash
python builder.py
```

### Основное меню
1. Список готовых моделей
2. Построить свою модель
3. Помощь
4. Выйти

### Сборка модели
1. Выберите датасет:
   - 1: MNIST (цифры 28x28, 1 канал)
   - 2: CIFAR-10 (изображения 32x32, 3 канала)

2. Соберите архитектуру:
   - Добавьте сверточные слои (Conv2D)
   - При необходимости добавьте MaxPooling2D
   - Завершите слоями Flatten и Dense

3. Обучите модель:
   - Укажите количество эпох (рекомендуется 5-15)
   - Выберите размер батча (рекомендуется 32)

4. Сохраните результат (модель сохранится в папку ptm)

---

## Готовые модели
В комплекте поставляются 6 предобученных моделей:
- `mnist_lite.keras` - базовая модель для MNIST
- `mnist_default.keras` - оптимальная модель для MNIST
- `mnist_large.keras` - продвинутая модель с residual-блоками
- `cifar_lite.keras` - легкая модель для CIFAR-10
- `cifar_default.keras` - сбалансированная модель для CIFAR-10
- `cifar_large.keras` - мощная модель для CIFAR-10

Все модели находятся в папке `ptm/`.

## Тестирование
Для тестирования на своих изображениях:
1. Поместите файлы в папку `images/`
2. Запустите `tester.py`
3. Выберите модель и режим тестирования
4. Система автоматически преобразует изображение под формат сети и покажет результат

Поддерживаемые форматы: PNG, JPG, JPEG.

## Как создавать модель
1. Выберите датасет  

    MNIST: цифры 28×28, 1 канал   
    CIFAR-10: цветные картинки 32×32, 3 канала


2. Начните со свёрток (Conv2D) - 2-3 слоя  

    Фильтры: начните с 32 → 64 → 128 (по нарастающей)  
    Шаг (strides): почти всегда 1 (сохраняет размер изображения)


3. Добавьте MaxPooling2D - 1-2 раза  

    Размер пула: 2 → уменьшает карту признаков вдвое  
    Пример: после свёрток добавьте один MaxPooling2D(2)


4. Завершите обработку  

    Flatten — обязателен перед Dense-слоями  
    Dense — 1–2 скрытых слоя:  
        Нейроны: 16–128 (рекомендуется 32 или 64)  
        Активация: ReLU
    Выходной слой:  
        Обязательно 10 нейронов  
        Активация: softmax


5. Советы  

    Не делайте больше 2 Dense-слоев после Flatten  
    Избегайте >128 нейронов в Dense — риск переобучения  
    Для MNIST часто хватает:
    Conv2D(32) → Conv2D(64) → MaxPooling2D(2) → Flatten → Dense(32) → Dense(10)
## Советы
- Начинайте с датасета MNIST - он проще и быстрее обучается
- Для CIFAR-10 рекомендуется использовать GPU
- Не устанавливайте слишком большое количество эпох во избежание переобучения
- Сохраняйте модели в формате `.keras` вместо устаревшего `.h5`
